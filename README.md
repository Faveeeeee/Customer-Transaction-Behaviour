
# Customer Transaction Behavior Analysis 
# Project Overview

This project analyzes customer transaction data to understand spending behavior, engagement patterns, and revenue concentration in a digital banking context. The goal is to identify high-value customers, assess product performance, and support data-driven decisions for growth and retention.

#  Tech Stack

Python: Pandas, 

SQL: MySQL

Tools: Jupyter Notebook, Git, GitHub

#  Methodology

Cleaned and validated customer, transaction, and product data using Python.

Designed a relational database schema and executed SQL queries to answer business-focused questions.

Analyzed customer value, transaction trends, inactivity risk, account behavior, and product adoption.

Interpreted results with a strong focus on business impact, not just metrics.

#  Key Insights

A small group of customers accounts for a disproportionately high share of transaction volume.

Premium account holders transact less frequently but contribute significantly higher value per transaction.

Investment and savings products are the strongest drivers of customer engagement.

Transaction activity shows sustained growth over time, indicating increasing platform usage.

#  Business Value

The insights from this analysis can support:

Targeted retention strategies for high-value customers

Product prioritization based on engagement impact

Better planning for growth and operational efficiency
